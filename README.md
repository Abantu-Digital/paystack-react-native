# paystack-react-native

A React Native wrapper for the Paystack iOS and Android SDK's

## Installation

```sh
npm install paystack-react-native
```

## Usage

```js
import { multiply } from 'paystack-react-native';

// ...

const result = await multiply(3, 7);
```
